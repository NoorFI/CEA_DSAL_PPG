Library System

A Python-based library management system using an AVL tree for cataloging books and a hash table for managing members.
It supports book borrowing, returning, searching, and reporting, with proper handling of stock and user limits.

The provided testss.py script covers:
- Adding members and books
- Searching by ISBN, title, or author
- Borrowing and returning books
- Borrow limit enforcement
- Reporting and edge cases
